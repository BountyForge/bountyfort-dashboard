# Contributing to BountyFort Dashboard

We welcome contributions to the BountyFort Dashboard!

## Local Setup

1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Run tests: `npm test`

## Coding Standards

- Use idiomatic TypeScript and strict typing.
- Keep components small and reusable.
- Follow the established design system (Tailwind + shadcn/ui).
- Add tests for new features or bug fixes.

## Pull Request Process

1. Create a feature branch.
2. Implement your changes.
3. Ensure all tests and lints pass.
4. Submit a PR with a clear description of your work.

## Accessibility

We strive to make BountyFort Dashboard accessible to everyone. Please ensure new components follow ARIA best practices and maintain good color contrast.
