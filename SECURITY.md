# Security Policy

## Wallet Safety

- **Transaction Review**: Always review the transaction details (amount, destination, contract function) in your wallet before signing.
- **Private Keys**: BountyFort Dashboard never asks for your private keys or recovery phrase.

## Environment Variables

Ensure that sensitive environment variables are not committed to version control. Use the provided `.env.example` as a template.

## Responsible Disclosure

If you find a security vulnerability in the dashboard, please report it privately to the maintainers. Do not open public issues for security-related bugs.

## Disclaimer

This dashboard is provided "as is". While we aim for high security standards, the final responsibility for transaction verification lies with the user. These tools have not been audited.
