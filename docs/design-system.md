# Design System

BountyFort Dashboard follows a clean, modern, and accessible design system based on Tailwind CSS and shadcn/ui.

## Typography
- **Primary Font**: Inter (Sans-serif)
- **Headings**: Bold, tracking-tight.
- **Body**: Regular, legible line height.

## Color Palette
- **Primary**: Blue (#3b82f6) - Used for primary actions and highlights.
- **Secondary**: Slate - Used for backgrounds and neutral elements.
- **Destructive**: Red - Used for critical actions like cancellation.
- **Muted**: Light Gray - Used for helper text and subtle borders.

## Components
- **Buttons**: Rounded corners, consistent padding, clear hover/active states.
- **Cards**: Subtle borders, rounded corners, consistent shadow for depth.
- **Badges**: Pill-shaped, used for statuses and categories.
- **Inputs**: Clear focus rings, consistent height.

## Layout
- **Grid**: 12-column responsive grid.
- **Spacing**: 4px base unit (Tailwind default).
- **Sidebar**: Fixed on desktop, collapsible on mobile.

## Accessibility
- Semantic HTML tags.
- High contrast ratios for readability.
- Focus visible indicators for keyboard navigation.
- ARIA labels where appropriate.
