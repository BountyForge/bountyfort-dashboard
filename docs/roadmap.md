# Roadmap

## Future Enhancements

### Phase 1: Core Polish
- [ ] Real-time event indexing for instant UI updates.
- [ ] Multi-wallet support (Albedo, Rabe, etc.).
- [ ] Rich text support for bounty descriptions (Markdown).

### Phase 2: Advanced Features
- [ ] Milestone-based bounty flows and partial payouts.
- [ ] Dispute resolution dashboard for third-party mediators.
- [ ] Contributor reputation profiles and achievement badges.

### Phase 3: Analytics & Scale
- [ ] Detailed maintainer analytics (spend, engagement, time-to-payout).
- [ ] Notification preferences (email/browser alerts for new submissions).
- [ ] Advanced filtering and saved search views.

## Ecosystem Integration
- [ ] Integration with Stellar indexers (e.g., Mercury) for historical data.
- [ ] Deep links to Stellar explorers for transaction verification.
